# Secure Data Classification and Labeling System

## Sub Title-
Role-Based Secure File Sharing with Automated Sensitive Data Classification and Access Control

## Description
This project provides a secure file sharing platform where uploaded files are automatically classified into sensitivity labels such as Public, Internal, Confidential, and Restricted.  
Access to files is controlled using Role-Based Access Control (RBAC) to prevent unauthorized users from viewing sensitive data.

## Features
- User Login and Authentication
- Role-Based Access Control (RBAC)
- File Upload and Download
- Automated Sensitive Data Classification
- Sensitivity Labeling (Public, Confidential, etc.)
- Audit Logging

## Tech Stack (Planned)
- Backend: FastAPI (Python)
- Frontend: React
- Database: SQLite (initially)
